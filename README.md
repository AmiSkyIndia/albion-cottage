# Albion Cottage — Static Website

Official website for **Albion Cottage**, a serene garden homestay in New Chandigarh, India.

## Overview

A fully static, single-page website built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no build tools required. Just open in a browser or serve from any web server.

## Project Structure

```
albion-cottage/
├── index.html        # Main single-page site
├── css/
│   └── style.css     # All styles (custom properties, responsive)
├── js/
│   └── main.js       # Sticky nav, mobile menu, scroll animations
└── README.md
```

## Sections

| Section | Description |
|---|---|
| Hero | Full-viewport banner with CTA buttons |
| Stats | Key highlights — rating, bedrooms, distance, host |
| About | Property overview, host intro, check-in/out times |
| Rooms & Spaces | 2 bedrooms, living room, kitchen, private garden |
| Amenities | Essentials, kitchen, room comforts, outdoor, host services |
| Gallery | Photo grid (swap placeholders with real images) |
| Location | Embedded map, address, nearby attractions |
| Book | Links to Airbnb, MakeMyTrip, Goibibo + contact buttons |

## Getting Started

No build step needed. Serve from WAMP or any local server:

```
http://localhost/albion-cottage/
```

Or open `index.html` directly in a browser.

## Customisation Notes

- **Gallery images** — replace the emoji placeholder `<div class="gallery-placeholder">` blocks with real `<img>` tags pointing to your photos.
- **Fonts** — loaded from Google Fonts (Playfair Display + Inter). Works offline if you self-host the font files.
